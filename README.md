# Program szkoleniowy: Nauczyciele wobec uczniów o niskiej motywacji

> Kompletny pakiet materiałów do wdrożenia programu szkoleniowego dla nauczycieli szkół zawodowych i techników.

## Cel programu

Przygotowanie nauczycieli do skuteczniejszej pracy z uczniami szkół zawodowych i techników, którzy:
- mają niską motywację do nauki przedmiotów ogólnokształcących,
- pochodzą z rodzin o ograniczonych zasobach edukacyjnych,
- doświadczają trudności społecznych lub emocjonalnych,
- prezentują niską wiarę we własne możliwości,
- nie dostrzegają związku między edukacją ogólną a przyszłym życiem zawodowym.

## Główne przesłanie

> Nie chodzi o to, aby wymagać mniej.  
> Chodzi o to, aby większa liczba uczniów była w stanie osiągnąć wymagane cele edukacyjne.

## Jak korzystać z pakietu

1. Zacznij od `00_strategia_wdrozeniowa.md` — daje szeroki kontekst i plan wdrożenia.
2. `01_program_szkolenia.md` to szkielet całego szkolenia — użyj go do planowania.
3. `02_scenariusze_warsztatow.md` zawiera gotowe materiały dla prowadzącego.
4. `03_narzedzia_dla_nauczycieli.md` to materiały rozdawane uczestnikom.
5. `04_argumenty_i_bibliografia.md` przyda się przy wstępnym przekonywaniu kadry i zawiera pełną bibliografię.
6. `_materialy_zrodlowe/badania_naukowe.md` to szczegółowe zaplecze badawcze dla każdej tezy.
